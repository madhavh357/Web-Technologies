        MySQL works with PHP because it has two extensions that work with MySQL: MySQLi and PDO. These extensions are drivers that allow PHP to get access to MySQL, and they also protect users from SQL injection attacks. MySQLi in particular provides a procedural interface to use SQL statements, which allows PHP to access the database. No PHP does not only work with MySQL. PHP can connect with MongoDB drivers which is different from MySQL because it is a non-SQL database program. It is important to learn MySQL and PHP for web development. MySQL runs with very little resource use, is simple to install and learn, and it is free. It is also open source for people to change.  I believe that because of how easy MySQL is to learn and how well it integrates with PHP, it is necessary to learn both for web development. Of course, if someone only wants to make a static blog or something it may not be necessary, but now, almost every website has to store some kind of data for some reason. MySQL is the simplest and easiest solution for that.

                  PHP connects with MySQL databases in two ways: MySQLi and PDO (PHP Data Objects). MySQLi is a specific driver and has an OOP-style syntax which is a bit easier than PDO. PDO, on the other hand, is a more general driver that works with 12 different databases. A PHP connection code (our config.php file) can ensure a tie with the database so on the server side, information from the database can be retrieved.  There are PHP statements which are built-in functions that allow you to select databases and perform queries, all of which allow you to display and manipulate data from a MySQL database.    While PHP and MySQL work well together for dynamic applications, PHP can stand on its own (similar to what we did for the IMSALookUp exercise). But, PHP does not need to rely on MySQL or anything because it can also work with JSON objects, which are key-value pairs in between curly brackets where keys are strings and values are a data type.    The combination PHP and MySQL are honestly one of the best ways to hone web development in a shorter amount of time. Compared to other languages, learning and utilizing PHP is much quicker and can produce dynamic webpages much more easily. MySQL is also a leading relational database and even without a heavily advanced knowledge of each type, one can make a website using the two that goes beyond simple HTML pages.

                                    PHP works with MySQL by opening a connection to the server where it can access databases of information to view, update, or otherwise manipulate. It connects to the server by using mysqli_connect, and it closes the connection using mysqli_close. PHP does not only work with MySQL. It has other packages like PDO (PHP Data Objects) that connect to other database systems. It is important to learn MySQL and PHP for web development. PHP is very widely used and it is a powerful tool for server side development. In addition, MySQL very common in websites, and it works well with PHP. Learning these can lead to effective and efficient websites. MySQL does not have a steep learning curve, as its syntax is similar to English.
